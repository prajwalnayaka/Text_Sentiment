### Run pip install requirements.txt in terminal

### Open terminal in your IDE and type "flask run" and hit enter

### Click the localhost link (http://127.0.0.1:5000)





This project uses the DistilBERT model. I concatenated 3 smaller datasets into a huge dataset in order to prevent data starvation as 
transformer architecture (such as in BERT) is extremely data hungry. Click [here](https://www.kaggle.com/datasets/prajwalnayakat/text-emotion) for the dataset.
